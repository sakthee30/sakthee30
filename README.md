## 🚀 Featured Projects

### 🔍 [Research Assistant Agent — LangGraph](https://github.com/sakthee30/research-assistant-agent)
> Agentic AI system that autonomously searches the web and reasons over results using the ReAct pattern

- Built with **LangGraph** implementing a stateful three-node graph — Researcher, Reasoner, and Responder
- **ReAct (Reason + Act) pattern** — agent decides whether to search again or respond based on result quality
- **Conditional edge** loops back to search if LLM finds results insufficient, with a 2-iteration cap to prevent infinite loops
- **Tavily Search API** for real-time web search + **Groq API (LLaMA 3.1)** as the reasoning LLM
- Exposed via **FastAPI REST endpoint** with Pydantic schemas and Swagger UI documentation

`Python` `LangGraph` `LangChain` `Groq API` `LLaMA 3.1` `Tavily Search` `FastAPI` `Pydantic` `Agentic AI`

---

### 📄 [Document Q&A System — LangChain RAG](https://github.com/sakthee30/document-qna-langchain)
> Production-style RAG backend API with triple LLM support — Groq, Ollama, and HuggingFace

- Built using **FastAPI + LangChain LCEL** with session-based conversation memory for multi-turn Q&A
- **ChromaDB** as persistent vector store with **HuggingFace Embeddings** (all-MiniLM-L6-v2)
- **Triple LLM support** — Groq API (cloud) + ChatOllama (local) + HuggingFace Inference API — switchable via single API parameter
- Complete RAG pipeline: PyPDFLoader → RecursiveCharacterTextSplitter → Embeddings → ChromaDB → LLM
- **Dockerized** with clean service architecture (PDF loader, vector store, RAG chain, memory)

`Python` `FastAPI` `LangChain` `ChromaDB` `Groq API` `HuggingFace` `Mistral-7B` `LLaMA3` `Docker` `RAG`

### 📄 [Document Q&A RAG System — FAISS Local Inference](https://github.com/sakthee30/document-qna-rag-local)
> Privacy-first full-stack RAG pipeline with local LLM inference — no data sent to external APIs

- Full-stack with **Streamlit** frontend + **FastAPI** backend
- Custom **FAISS vector store** with **SentenceTransformers** (all-MiniLM-L6-v2) for embeddings
- Local **Mistral LLM** via Ollama — privacy-first, fully offline inference
- Complete RAG pipeline: pypdf → Text Chunking → SentenceTransformers → FAISS → Semantic Search → Mistral

`Python` `FastAPI` `Streamlit` `FAISS` `SentenceTransformers` `Mistral` `Ollama` `RAG`
