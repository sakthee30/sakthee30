## ЁЯЪА Featured Projects

### ЁЯФН [Research Assistant Agent тАФ LangGraph](https://github.com/sakthee30/research-assistant-agent)
> Agentic AI system that autonomously searches the web and reasons over results using the ReAct pattern

- Built with **LangGraph** implementing a stateful three-node graph тАФ Researcher, Reasoner, and Responder
- **ReAct (Reason + Act) pattern** тАФ agent decides whether to search again or respond based on result quality
- **Conditional edge** loops back to search if LLM finds results insufficient, with a 2-iteration cap to prevent infinite loops
- **Tavily Search API** for real-time web search + **Groq API (LLaMA 3.1)** as the reasoning LLM
- Exposed via **FastAPI REST endpoint** with Pydantic schemas and Swagger UI documentation

`Python` `LangGraph` `LangChain` `Groq API` `LLaMA 3.1` `Tavily Search` `FastAPI` `Pydantic` `Agentic AI`
